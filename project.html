Intelligent Smart Attendance Monitoring & Payroll Generation

This project leverages Machine Learning to automate staff attendance tracking and payroll generation. It integrates facial recognition for accurate attendance, calculates Internal Assessment (IAT) marks for students, and generates payroll reports for faculty based on attendance data.

Key Features:

  • Automated attendance using ML-based facial recognition.
  • Real-time monitoring dashboard for administrators.
  • Payroll generation linked directly to attendance records.
  • Student IAT marks calculation integrated into the system.
  • Secure database management with SQL.

Tech Stack: Python, Django, scikit-learn, OpenCV, SQL, HTML, CSS, JavaScript.

Project Value: This project demonstrates end‑to‑end integration of AI and web development — combining facial recognition, data analytics, and payroll automation in a single platform. It highlights your ability to build intelligent systems that improve efficiency and accuracy in academic administration.

Royal Student Management Dashboard – Django Web Application

The Royal Student Management Dashboard is a full‑stack Django web application designed to simplify academic data handling with a premium indigo‑gold themed interface. It provides administrators with real‑time insights into student records, departmental statistics, and attendance tracking, while ensuring a consistent and elegant user experience across all pages.

Key Features:

  • Dynamic dashboard displaying total students, departments, and attendance.
  • Interactive charts (Chart.js) for student distribution per department.
  • Complete CRUD operations for student records (add, edit, delete, view).
  • Integrated IAT marks calculation and reporting for students.
  • Reusable template system with base.html inheritance for consistency.
  • Royal indigo‑gold UI theme with gradient backgrounds and animations.
  • Responsive design ensuring accessibility across devices.

Tech Stack: Python, Django, SQLite, HTML, CSS, JavaScript (Chart.js).

Project Value: This project demonstrates the integration of backend logic and frontend design, showcasing your ability to build professional web applications that combine data management, visualization, and premium UI/UX design. It highlights your skills in Python, Django, and modern web development practices.
